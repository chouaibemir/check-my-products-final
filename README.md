## Project structure
The `app` folder holds the files for node modules (controllers, services, errors...) <br />
The `public` folder holds the files for the front-end angular application <br />

## How to run the server
1- Open a new terminal
2- Navigate to app folder with `cd app` <br />
3- Run `npm install` to install dependencies <br />
4- Run `nodemon` to run the server (or `node server.js`) <br />

## How to run the client
1- Open a new terminal
2- Navigate to client folder with `cd public` <br />
3- Run `npm install` to install dependencies <br />
4- Run `ng serve`. <br />
5- Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files. <br />
